JAVA——String

发表于 2020-04-17 Valine：
String字符串
* 创建方式
    * pbulic String("")
    * public String(char[] array)
    * public String(byte[] array)
    * ="";
字符串常量池
* String s = " "进行赋值的对象存放在字符串常量池
* 字符串常量池的对象存放在堆区
* 也就是说字符串常量池内存放的还是地址
字符串比较
* ==比较的是引用的地址。
* a.equals(b);重写了后比较的是字符串。
* a.equalsIgnoreCase(b);忽略大小写比较
字符串比较
* str.length();//获取str长度
* str1.concat(str2);//拼接字符串
* str.charAt(index);//返回索引位置的char
* str.indexOf(strsub);//返回字串首次出现的位置
字符串截取
* str.substring(index);//从index到末尾
* str.substring(begin,end);//从begin到end
字符串转换
* char[] ch = str.toCharArray();
* byte[] b = str.getBytes();
* String s = str.replace(oldstr,newstr);
字符串分割
* String[] ss = str.split("//.");// "//."表示.

StringBuilder

* String底层是一个被final修饰的数组，不能改变
* 每次产生新的字符串都会在内存开辟一块空间

StringBulider：字符串缓冲区，可以提高操作效率，底层也是数组，但是没有被final修饰。
如果超出预期空间会自动扩容。
