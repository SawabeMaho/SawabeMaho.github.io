JS是什么

1. 脚本语言：不需编译，直接运行
2. 解释性语言：遇到一行，解释一行
3. 动态类型语言：不需要限定变量类型
1.动态类型：动态类型语言是指在运行期间才去做数据类型检查的语言。 Python
2. 静态类型：静态类型语言是指数据类型是在编译期间检查的。  C++
4. 面向对象语言
5. 弱类型语言
1. 强类型是指不允许隐式变量类型转换 java
2. 弱类型则允许隐式类型转换 c++ js ---
