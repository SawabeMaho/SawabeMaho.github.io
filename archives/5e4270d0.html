Java常用API与数据结构

发表于 2020-03-20 更新于 2020-03-29 分类于 jAVA Valine：

Object类

Object类是所有类的祖先类，我们认为一个类如果没有继承，默认父类为Object类。

对象.toString()

Object类中有一个toString方法，默认是打印对象的地址。

对象1.equals(对象2)

基本数据类型比较值是否相等
引用数据类型比较地址是否相等

类Date

java.util.Date:表示日期和时间的类，精确到毫秒

抽象类DataFormat

格式化日期返回一个字符串，由于是抽象类，所以必须使用其子类。

StringBuilder

由于String是一个常量，不可修改每次改变都会在字符池中创建一个新的String对象，浪费空间，JAVA提出了StringBuilder字符缓冲区。

包装类

所谓包装类就是为了将4类8中基本类型进行某种改造，封装一些成员方法能够对其进行操作。
4类8种基本类型也有8种包装类。
虽然包装类有一些构造函数，但是为了方便，可以把包装类当中基本类型来使用(自动装箱，自动拆箱)
有很多方法建议使用时查找文档。

Collection集合

集合的长度是可变的，数组的长度是固定的。 集合可以存储对象，数组可以存储任意类型(包括对象).

Collection是所有单列集合的最顶层接口，任意单列集合都是此接口的实现类。可以实现多态。
共用方法集合：
add添加一个元素
clean清空
remove删除
contains判断元素是否存在
size返回元素个数
toArray把集合元素存入数组

迭代器

增强for

所有的单列集合和数组都支持foreach

范型

JAVA范型是一个实现未知数据类型传递发策略，与C++模板方法类似，不同之处在于JAVA是运行时确定数据类型，C++是编译时确定数据类型。
E - Element (在集合中使用，因为集合中存放的是元素)
T - Type（Java 类）
K - Key（键）
V - Value（值）
N - Number（数值类型）
? - 表示不确定的java类型

范型通配符

List

List 继承自Collection，有序，带索引，允许重复。
除了继承的Collection方法还有自己的一些方法。

ArrayList

ArrayList底层使用数组实现，查询快，增删慢。后面会详细分析其底层实现。

LinkedList

LinkedListd底层是链表，查询慢，增删快，包含很多操作首尾元素的方法（特有，不能使用多态,但继承的方法一直可以使用）。

vector

1.2版本后被ArrayList取代，建议使用ArrayList。是同步的(单线程)，ArrayList是异步的。

Set

Set是一个数学上的Set抽象，不能包含重复元素。确切的讲是不包含obj1.equals(obj2)==true;的元素，Set是无序的。只能有一个null；
我们知道父类是共性抽取，Set无索引，List有索引，所有Collection的所有方法都是无序的，Set可以直接使用。

HashSet

此类实现Set接口，由哈希表支持实际上是一个HashMap，只能有一个null；

  1. 不允许重复元素
  2. 没有索引
  3. 无序
  4. 底层哈希，查询很快。
    哈希表是一个数组+链表或者数组+红黑树（哈希冲突超过8位）。
    原理：
    先HashCode，得到一个值hsd，查找hsd对应set中是否有值，无则插入，有则调用equals比较此时此处的值与要存的值进行比较，如果不相同则插入以hsd开通的链表或红黑树。
    很明显如果需要存储自定义类需要重写先HashCode(默认hash对象地址)和equals(默认比较对象地址)；
LinkedHashSet

继承自HashSet，与HashSet使用类似，但是是有序的。

可变参数

方法的参数类型已经确定，但是个数不同([0~n])，可以使用可变参数。
一个方法只能有一个可变参数，且放在参数列表末尾。

Collections对Collection子类进行操作。

Map集合

Map包含一对键值对的类，将键映射到值的对象，一个映射不能包含重复的键，每个键最多映射一个值，类似与Python中的字典。 特点： 双列集合 key不允许重复 key-value是一一对应的。 #### HashMap 底层哈希表，数组+链表或数组+红黑树，无序，异步。 遍历，keyset会返回key，遍历set通过key遍历map;

LinkedHashMap

继承自HashMap哈希表+链表：有序，存取顺序一致。

Hashtable

早期版本的HashMap,建议使用HashMap，同步，不能存储null。
