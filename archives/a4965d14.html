Linux复习1--基本指令

发表于 2020-02-23 更新于 2020-03-01 分类于 Linux Valine：

Linux目录结构

linux的核心思想：万物皆文件。

vi/vim使用指南。

三种模式

  • 正常模式： 打开一个文档后的默认模式，可以使用上下左右。
  • 插入模式： 插入模式按下i后的模式。
  • 命令行模式： 插入模式按下ESC后回到正常模式，正常模式下按下： 后 可以wq，q。

    快捷键：

正常模式下：

  • yy： 复制当前行，5yy 复制当前及下共5行
  • dd： 删除当前行，5dd
  • /单词： 查找单词
  • gg: 首行行首
  • G: 尾行行首
  • u： 撤销操作，类似word的上一步。
  • 20 shift+g 到第20行行首

命令行模式下：

  • set nu 添加行号
  • set nonu 取消行号

关机：

  • shutdown 关机

    • shutdown -h now 立即关机
    • shutdown -h 1 1分钟后关机
    • shutdown -r now 立即重启

  • half 关机

  • reboot 重启

  • syn 把内存的数据写入磁盘，建议关机前先syn。

  • logout 注销(图像下无效)

添加用户

  1. Linux是多用户多任务操作系统，任何一个要使用系统资源的用户，首先要向系统管理员申请一个账号，然后以账号身份进入操作系统。
  2. Linux用户至少属于一个组。
  3. 每个用户都对应一个/home/用户名文件夹。即使用当前身份进入系统后的默认文件夹。
  • useradd [选项] 用户名
  • useradd -d /home/目录X 用户名 将强制是用户名的家目录为目录X
  • passwd 用户名 给用户添加/修改密码

删除用户

  • userdel 用户名 删除用户并保留家目录
  • userdel -r 用户名 删除用户并删除家目录

查看用户信息

切换用户

  • su - 用户名 (exit退出切换)

组操作

  • groupadd 组名 添加组
  • groupdel 组名 删除组
  • useradd -g groupname username 添加用户并将用户归入group。
  • usermod -g groupname username 修改用户操作组

组信息

/etc/passwd 用户配置文件
/etc/group 组配置文件
/etc/shadow 口令配置文件加密
