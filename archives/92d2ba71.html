生产者消费者

发表于 2020-04-14 更新于 2020-04-15 分类于 JAVA Valine：

最近面试碰到了生产者消费者模型，第一次没写出来，然后又碰到了。
记录一下。
模型中有三个关键的对象。
生产者，消费者，以及消费对象。
从名字也能看出都是干嘛的。
生产者：包子铺
消费者：吃货
消费对象：包子

包子就是一个对象，没什么好说的，必须有一个状态flag表示还有没有包子，为了方便我们设为boolean，true表示有包子，false表示没有包子。
int也行，flag>1表示可以卖包子，生产者等待，flag<1表示没有包子，消费者等待。
包子铺是生产者，为了实现线程安全，必须锁住包子，锁住之后去判断有没有包子，有包子，等待，没有包子，生成包子，生成后唤醒消费者。
吃货是消费者，为了实现线程安全，也必须锁住包子，锁住之后去判断有没有包子，有包子，吃，没有包子，等待。
