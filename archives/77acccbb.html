查找指令

find指令 find [查找范围] [选项] “str”
  1. find /root -name hello.txt 查找/root目录下hello.txt 文件
  2. find /home -user pain 查找/home目录下所有属于pain的文件
  3. find / -size +20M 查找/ 根目录下所有大于20Mb的文件 +大于 -小于 不写 等于
locate指令 快速定位文件路径
  1. locate a.txt 输出a.txt的路径
  2. 不知道为什么我的linux不能使用这条指令
grep指令 和 管道符号|
  1. grep 过滤查找 grep [选项] “str” 源文件
  2. grep -n 显示行号
  3. grep -i 忽略大小写
  4. grep -ni “yes” hello.txt 查找hello.txt中出现yes的行。
  5. | 将前一个命令的输出处理结果传递给后面的命令处理
  6. cat hello.txt | grep -ni yes 查找hello.txt出现yes的行

解压缩指令

gzip [filename] / gunzip [filename]
  1. gzip a.txt ==>a.txt.gz 替换源文件
  2. gunzip a.txt.gz ==>a.txt
zip [选项] xxx.zip [filename] /unzip [选项] [filename]
  1. zip -r file.zip /home/pain/test/ 将/home/pain/test/目录下所有文件压缩为file.zip
  2. unzip -d /home/pain/test2 file.zip 将file.zip解压到指定目录
tar [] xxx.tar
  1. -c 产生.tar 打包文件
  2. -v 显示详细信息
  3. -f 指定压缩后的文件名
  4. -z 打包同时压缩
  5. -x 解压.tar文件

tar -zcvf a.tar.gz a1.txt a2.txt 把a1.txt，a2.txt 压缩为a.tar.gz
tar -zxvf a.tar.gz 将a.tar.gz 解压到当前目录
tar -zxvf a.tar.gz -C 指定目录 解压到指定目录

组管理

linux中
用户必须属于一个组
文件有 所有者 所在组 其他组 三种权限
假如文件由小明创建，则默认所有者为小明，所在组为小明所在组，其他组为除小明所在组之外的组。
ls -al 会显示出所有者和所在组
chown 用户名 文件名 修改文件所有者
chgrp 组名 文件名 修改文件所在组
chown 用户名：组名 文件名 即修改文件所有者也修改文件所在组

usermod -g 组名 用户名 修改用户所在组
usermod -d 目录 用户名 修改用户初始登录目录

权限管理

-rwxrwxrwx
文件所有者权限
文件所在组权限
其他组权限
-：表示普通文件
d：目录
l：软连接
c:字符设备-鼠标，键盘
b：块文件-硬盘
r：读
w：写
x：执行

修改权限

chmod
第一种方式 u g o a + - =
chmod u=rwx filename 设置user 权限为rwx
chomd u+x filename 数字user 权限 + x
chomd a-x filename ugo 权限-x

第二种方式
chmod 751 文件名
