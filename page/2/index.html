异常

异常是指程序运行的过程中，出现的非正常情况，最终会导致JVM非正常停止。
在JAVA中，异常本身是一个类，产生异常是指创建一个异常对象并抛出。
异常处理的方式就是中断处理。
异常的根类是Throwable，下述两个子类Error和Exception。
异常还有一个子类叫运行时异常。

操作系统复习

发表于 2020-03-11 更新于 2020-03-24 分类于 操作系统 Valine：

操作系统的特性

  1. 并发：同一段时间内多个程序执行
  2. 共享：系统的资源可以被多个并发的进程或线程共用
  3. 虚拟：通过时分复用或者空分复用，把一个物理实体虚拟为多个
  4. 异步：系统中的进程总是以不可预知的速度推荐

操作系统的主要功能

  1. 进程管理
    进程控制、进程同步、进程通信、进程调度
  2. 内存管理
    内存分配、内存保护、地址映射、内存扩充
  3. 设备管理
    管理所有外围设备
